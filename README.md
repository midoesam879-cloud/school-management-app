# نظام إدارة المدرسة - School Management System

تطبيق Flutter شامل لإدارة المدارس مع دعم كامل للمدير والمعلمين والطلاب.

## المميزات الرئيسية

### 1. نظام المستخدمين
- **المدير**: التحكم الكامل في النظام
  - إضافة وحذف المعلمين
  - إضافة وحذف الطلاب
  - إنشاء الفصول الدراسية
  - إدارة الجدول الدراسي
  - إدارة لوحات رؤساء المواد والمعامل والأخصائيين

- **المعلم**: إدارة الفصول والدرجات
  - عرض الفصول المسندة
  - إدارة درجات الطلاب
  - تسجيل الحضور والغياب

- **الطالب**: عرض البيانات الأكاديمية
  - عرض الجدول الدراسي
  - عرض الدرجات
  - عرض سجل الحضور والغياب

### 2. قاعدة البيانات
- استخدام SQLite للتخزين المحلي
- 6 مواد دراسية محددة مسبقاً:
  - عربي
  - حساب
  - انجليزي
  - دين
  - علوم
  - دراسات

### 3. لوحات التحكم المتخصصة
- لوحة رؤساء المواد
- لوحة المعامل
- لوحة الأخصائي الاجتماعي

## متطلبات التثبيت

### المتطلبات الأساسية
- Flutter SDK (إصدار 3.0 أو أحدث)
- Dart SDK (يأتي مع Flutter)
- Android Studio أو Xcode (حسب نظام التشغيل)

### التثبيت على Windows/macOS/Linux

1. **تثبيت Flutter**
   ```bash
   # قم بتحميل Flutter من:
   # https://flutter.dev/docs/get-started/install
   
   # تحقق من التثبيت
   flutter --version
   ```

2. **استنساخ المشروع**
   ```bash
   cd school_management_app
   ```

3. **تثبيت الاعتماديات**
   ```bash
   flutter pub get
   ```

4. **تشغيل التطبيق**
   ```bash
   # على محاكي أو جهاز فعلي
   flutter run
   
   # أو لتشغيل على ويب
   flutter run -d chrome
   ```

## استخدام التطبيق

### كمدير

1. **تسجيل الدخول**
   - اختر "مدير" من الخيارات الثلاثة
   - أدخل بيانات الدخول الافتراضية

2. **إضافة معلم**
   - اذهب إلى "إضافة معلم"
   - أدخل: الاسم، اسم المستخدم، كلمة المرور، المادة
   - جميع الحقول إجبارية

3. **إضافة طالب**
   - اذهب إلى "إضافة طالب"
   - أدخل: الاسم، اسم المستخدم، كلمة المرور، الصف الدراسي
   - يمكن اختيار فصل (اختياري)

4. **إضافة فصل**
   - اذهب إلى "إضافة فصل"
   - أدخل: اسم الفصل، الصف الدراسي، السعة

5. **إدارة الجدول الدراسي**
   - اختر الفصل والمعلم والمادة واليوم والأوقات

### كمعلم

1. **تسجيل الدخول**
   - اختر "معلم"
   - أدخل بيانات دخول المعلم

2. **إدارة الدرجات**
   - اختر الفصل ثم الطالب والمادة
   - أدخل الدرجة

3. **إدارة الحضور**
   - اختر الفصل والمادة والتاريخ
   - حدد حالة كل طالب (حاضر/غائب/متأخر)

### كطالب

1. **تسجيل الدخول**
   - اختر "طالب"
   - أدخل بيانات دخول الطالب

2. **عرض الجدول الدراسي**
   - يعرض جميع الحصص المجدولة لفصلك

3. **عرض الدرجات**
   - يعرض جميع درجاتك مع المعدل

4. **عرض الحضور**
   - يعرض سجل الحضور والغياب مع النسبة المئوية

## ملاحظات مهمة

1. **التحقق من الحقول الفارغة**: لا يمكن ترك أي حقل مطلوب فارغاً
2. **أسماء المستخدمين الفريدة**: يجب أن تكون أسماء المستخدمين فريدة
3. **قاعدة البيانات المحلية**: جميع البيانات تُخزن محلياً على الجهاز
4. **اللغة العربية**: التطبيق مدعوم بالكامل للغة العربية
